# Peppard IPAS Operator Dashboard

Compliance and operations dashboard for Peppard Investments' 8 IPAS
accommodation centres. Dual regulatory axis: Department of Justice IPPS
contractual inspections (RAG findings, 14-day evidence loop) and HIQA
National Standards (40 standards / 10 themes) — one dataset feeds both.

Built by Origin Care Group. Sibling app to Genisis3 (connects-health-portal),
sharing its design system with Peppard branding.

```bash
npm install
npm run dev      # http://localhost:5184
npm run build
```

See `CLAUDE.md` for the full project reference and `docs/` for the
extracted client requirements and source data.
